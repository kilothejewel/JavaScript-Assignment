# JavaScript Assignment

A collection of JavaScript exercises covering fundamental programming concepts including conditional logic, type conversion, functions, objects, arrays, and loops.

## Features

### Question 1: Grade Calculator
A function that converts test scores to letter grades and determines pass/fail status.
- Grades: A (90+), B (80+), C (70+), D (60+), F (below 60)
- Uses `switch` statement for grade assignment
- Uses ternary operator for pass/fail determination

### Question 2: Type Conversion & Truthy/Falsy Values
Demonstrates type conversion using the `Number()` constructor and evaluates truthy/falsy values in JavaScript.
- Tests values: 0, "", "hello", null, undefined, NaN
- Uses `forEach()` to iterate through test values
- Conditional logic to determine truthiness

### Question 3: Greeting Bot
A simple function that returns personalized greetings based on time of day.
- Uses ternary operator for conditional greeting
- Takes `name` and `isMorning` parameters
- Returns appropriate greeting string

### Question 4: Social Media Post Object
Demonstrates object creation, methods, destructuring, and object manipulation.
- Object with properties: username, caption, likes, comments
- `addLike()` method to increment likes
- Object destructuring to extract properties
- Shows `this` keyword usage in methods

### Question 5: Array Operations & Loops
Three different exercises showcasing array spread operator and loop patterns.
- **Array Spread**: Combines two arrays using the spread operator
- **Star Pattern**: Generates a pyramid pattern using nested loops and string repetition
- **Countdown Loop**: While loop that counts from 10 down to 1

## Usage

Open `script.js` in a JavaScript environment (browser console, Node.js, etc.) and execute to see the output of all functions and exercises.

## Concepts Covered

- Conditional statements (switch, ternary operator)
- Functions and parameters
- Objects and methods
- Destructuring
- Array operations (spread operator)
- Loops (for, while, forEach)
- Type conversion and truthiness
- String interpolation
- Console logging
